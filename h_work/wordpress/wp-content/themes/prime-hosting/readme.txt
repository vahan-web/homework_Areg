== Description ==

Prime Hosting is an minimal WordPress with customizable features for web hosting and web server compnies, 
Which allows you to build a beautiful own web hosting company or web server company website in a few hours!

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.


== Known limitations ==
The primary menu supports 2 sub menu levels.
The secondary menu suports one level.
